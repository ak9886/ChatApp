#  Java Chat App (JDBC + PostgreSQL)

A lightweight **Java Swing-based Chat Application** that allows multiple users to **register, log in, and chat** in real time (via database polling).  
Built using **JDBC** for database connectivity and **PostgreSQL** as the backend database.

---

##  Features

-  User registration and login with password hashing (SHA-256)
-  Real-time chat between users (using periodic message refresh)
-  Dark mode UI built with Java Swing
-  Persistent message storage via PostgreSQL
-  Secure login system using prepared statements
-  Lightweight and runs locally

---

##  Tech Stack

| Component | Technology |
|------------|-------------|
| **Frontend (UI)** | Java Swing |
| **Backend (Logic)** | Java (Core + AWT + Event Handling) |
| **Database** | PostgreSQL |
| **Connectivity** | JDBC |
| **Hashing** | SHA-256 (MessageDigest) |
